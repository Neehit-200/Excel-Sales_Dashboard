# 🛒 Dmart Sales Dashboard | Excel Data Analytics Project

## 📌 Project Overview

The **Dmart Sales Dashboard** is an interactive Excel-based business intelligence project designed to analyze and visualize retail sales performance using the Dmart dataset.  

This dashboard provides meaningful insights into sales trends, customer behavior, product performance, profit analysis, and regional sales distribution through dynamic visualizations and interactive reporting features.

The project demonstrates practical implementation of Excel for real-world business reporting and data analytics.

---

# 🎯 Project Objectives

- Analyze overall sales and profit performance
- Identify top-performing product categories
- Track regional and customer segment sales
- Compare monthly and yearly business trends
- Build an interactive and user-friendly dashboard
- Generate actionable business insights from raw retail data

# 📂 Dataset Information

The dataset contains retail sales transaction records from Dmart including:

- Order ID
- Customer Name
- Product Category
- Sub-Category
- Region
- Sales
- Quantity
- Profit
- Order Date
- Payment Mode
